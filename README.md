# MarceeAI

Marketing/portfolio site for MarceeAI, an AI automation consulting practice built around n8n and Make workflows for ecommerce teams.

Single-page static site, no build step required.

## Structure

- `index.html` - the entire site (markup, styles, and script are all inline)
- `assets/` - favicon and logo images
- `Joshua_Novak_Resume.pdf` - resume, linked from the Resume section
- `robots.txt`, `sitemap.xml` - search engine crawling config

## Local preview

Serve the directory with any static file server, e.g.:

```
npx serve .
```

Then open the printed local URL in a browser.
